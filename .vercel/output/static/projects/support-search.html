SupportSearch

Chrome extension made to assist my department at Clubspeed in order to more quickly and efficiently retrieve customer information, as well as providing important links. Created in tandem with an admin site (the SupportSearch-Admin repo on github) to assist in some features that are very useful internally, as well as the API for the extension to use. The admin site also include a feature request system that allows for user voting! All data is hosted using Supabase Database.
